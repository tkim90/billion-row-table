- Frontend should use Typescript and Bun.
- Use NextJS as the frontend.
- Backend should use Rust and Actix Web.

Frontend code goes under ./frontend
Backend code goes under ./backend

# Frontend
- Use small, composable components. Try not to repeat the same divs over and over. If you see there are repeatable use cases, pull it out as a component.
- Use Tailwindcss as the styling engine.
